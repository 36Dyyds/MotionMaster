# MotionMaster

#### 介绍
MotionMaster（动作识别大师）是一款基于 Python 的入门级动作识别项目，旨在为用户提供简单易用的图片和视频动作识别功能。无论是初学者还是有一定编程经验的用户，都能够轻松上手，通过该项目快速了解和应用动作识别技术。
